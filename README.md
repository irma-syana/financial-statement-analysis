# Financial Statement Analysis

## Project Overview

This project demonstrates an end-to-end financial statement analysis workflow using Python. It covers data preparation, financial classification, departmental analysis, and management reporting visualization.

## Objectives

- Clean and transform general-ledger data
- Standardize account and department classifications
- Visualize revenue, expenses, and profitability
- Compare financial performance across periods
- Produce management-ready visualizations

## Tools

- Python
- Pandas
- Jupyter Notebook
- Matplotlib
- Plotly
- Microsoft Excel
